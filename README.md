# 🍛 MealPilot — Nigerian Meal Planner

> Your Nigerian kitchen companion for meal planning, budgeting, and cooking inspiration.

## Features

- **Weekly Meal Planner** — Plan breakfast, lunch & dinner for every day
- **50+ Nigerian Meals** — Full database with costs, ingredients, categories
- **Shopping List Generator** — Auto-generates from your plan, grouped by category
- **Budget Tools** — "What can I cook with ₦X?" for any amount
- **Budget Estimator** — Weekly household food cost calculator
- **What Can I Cook With What I Have?** — Ingredient matching engine
- **30 Full Recipes** — Step-by-step Nigerian recipes with cook times
- **Food Budget Tracker** — Track weekly & monthly food spending
- **Dark Mode** — Easy on the eyes at night
- **Favourites & Recents** — Save meals you love
- **Works Offline** — Everything stored in localStorage

## Deploy to GitHub Pages

1. Fork or upload these files to a new GitHub repo
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your app will be live at `https://yourusername.github.io/mealpilot`

## Files

```
mealpilot/
├── index.html   — App structure
├── style.css    — All styling (dark mode, mobile-first)

└── README.md    — This file
```

## Tech Stack

- Pure HTML, CSS, Vanilla JavaScript — no frameworks, no dependencies
- localStorage for all data persistence
- Works completely offline after first load
- Optimized for mobile (Nigerian working women, 25–40)

## Pricing Data

All costs are estimated in Nigerian Naira (₦) based on 2024 market prices.
Prices vary by location and season — the app is for guidance only.
